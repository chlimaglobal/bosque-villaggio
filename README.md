## 🔄 Como atualizar o site

1. Gere o novo `index.html` atualizado
2. Acesse este repositório no GitHub
3. Clique em `index.html` → ícone de lápis (editar)
4. Substitua o conteúdo → **Commit changes**
5. O Netlify detecta o commit e republica automaticamente em ~1 minuto
